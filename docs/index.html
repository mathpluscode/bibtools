bibtools

A bibliography toolkit for LaTeX — built as a Claude Code plugin.

GitHub

bibtidy cross-checks BibTeX entries against Google Scholar, CrossRef, and conference/journal sites. It upgrades arXiv/bioRxiv preprints to published versions (even when the title changed upon publication), corrects metadata (authors, pages, venues), and flags semantic duplicates.

Install

Add the marketplace in Claude Code:

/plugin marketplace add mathpluscode/bibtools

Install the plugin:

/plugin install bibtools@mathpluscode-bibtools

Reload plugins:

/reload-plugins
